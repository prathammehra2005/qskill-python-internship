##  1. Sales Data Analysis & Visualization

Analyze a retail sales dataset using **Pandas**, **NumPy**, **Matplotlib**, and **Seaborn** to extract business insights and create professional visualizations.

### Features

- Data Cleaning
- Statistical Analysis
- Average Sales by Region
- Average Sales by Category
- Scatter Plots
- Correlation Heatmap
- Business Recommendations

**Technologies**

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

#  Getting Started

### Clone the repository

```bash
git clone https://github.com/yourusername/qskill-python-internship.git
```

### Move into the project

```bash
cd qskill-python-internship
```

Each project contains its own `README.md` and `requirements.txt`.

Install the required libraries for the project you want to run:

```bash
pip install -r requirements.txt
```

Run the Python file:

```bash
python filename.py
```
