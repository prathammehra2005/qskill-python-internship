#  Matrix Operations Tool

An interactive command-line application built using **Python** and **NumPy** that performs common matrix operations such as addition, subtraction, multiplication, transpose, and determinant calculation.

This project was developed as part of the **QSkill Python Development Internship**.

---

##  Project Overview

The Matrix Operations Tool allows users to create matrices interactively and perform various mathematical operations through a simple menu-driven interface.

The application validates user input, formats matrix output neatly, and allows users to save matrices for reuse during the session.

---

##  Features

- Matrix Addition
- Matrix Subtraction
- Matrix Multiplication
- Matrix Transpose
- Determinant Calculation
- Save Matrices for Future Operations
- View Saved Matrices
- Input Validation
- Professional Matrix Formatting
- Menu-driven Interface

##  How It Works

1. Run the program.
2. Select an operation from the menu.
3. Enter matrix dimensions.
4. Input matrix values.
5. View the calculated result.
6. Save matrices for later use (optional).
7. Repeat operations or exit.

---

##  Available Operations

| Operation | Description |
|-----------|-------------|
| Addition | Adds two matrices of the same size |
| Subtraction | Subtracts one matrix from another |
| Multiplication | Performs matrix multiplication |
| Transpose | Computes the transpose of a matrix |
| Determinant | Computes the determinant of a square matrix |
| View Saved Matrices | Displays all saved matrices |
